# German Fashion Shop

## Project Overview
- Purpose: A modern, responsive storefront for German fashion, showcasing products with a fast, smooth browsing experience.
- Main Features:
  - Product catalog with categories and filtering
  - Global search overlay for quick lookup
  - Detailed product pages with images and descriptions
  - Shopping cart with persistent state and slide-out sidebar
  - Account page and basic auth modals
  - Polished UI with skeleton loading and motion effects
  - SPA routing with client-side navigation and 404-safe refresh behavior
- Target Users: German-speaking shoppers and small retailers seeking a clean, performant e-commerce front-end.

## Technologies
- Framework: React 18
- Routing: React Router 6
- State: Zustand
- Styling: Tailwind CSS, tailwind-merge
- Animations: Framer Motion
- Icons: Lucide React
- Language: TypeScript 5
- Build Tool: Vite 5 with @vitejs/plugin-react
- Tooling: ESLint (TypeScript + React Hooks), PostCSS, Autoprefixer
- Deployment: Vercel (SPA rewrites)
